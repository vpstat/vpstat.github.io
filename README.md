# Personal Profile Template

A modern, responsive template for your personal GitHub Pages profile.

## Getting Started

1.  **Add Your Content**:
    *   Open `index.html` in a text editor.
    *   Look for comments like `<!-- REPLACE WITH YOUR NAME -->` and `<!-- PASTE YOUR ABOUT CONTENT HERE -->`.
    *   Replace the placeholder text with your own information.

2.  **Add LinkedIn Badge**:
    *   Go to your LinkedIn profile.
    *   Click on "Edit public profile & URL" (usually on the right sidebar).
    *   Scroll down to the "Public Profile Badge" section.
    *   Copy the provided script code.
    *   Paste it into `index.html` inside the `<!-- PASTE YOUR LINKEDIN BADGE SCRIPT CODE HERE -->` section.

3.  **Customize Colors (Optional)**:
    *   Open `style.css`.
    *   Modify the variables in the `:root` section at the top to match your preferred color scheme.

## Deployment to GitHub Pages

1.  Commit and push these changes to your repository.
2.  Go to your repository settings on GitHub.
3.  Navigate to the "Pages" section.
4.  Under "Source", select "Deploy from a branch".
5.  Select your main branch (usually `main` or `master`) and the `/ (root)` folder.
6.  Click "Save".

Your site will be live at `https://<your-username>.github.io/<repo-name>/` shortly!
